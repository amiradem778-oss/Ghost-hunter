👻 Ghost Hunter License

This is a simple website made with HTML and CSS for a Grade 11 web design assignment.

About the Project

The website shows a fictional Ghost Hunter License. It explains the idea of a paranormal investigator permit and allows users to enter their information in a simple form.

What the Website Has

- A home page with the project title
- Short information about the ghost hunter license
- A form where a person can apply for the license
- A submit button
- A simple footer
